Objektorientierung mit Fantasy-Figuren

In diesem Beispiel haben unsere Figuren Attribute und Fähigkeiten. JavaScript-Klassen helfen uns, solche Strukturen einfach zu modellieren.

Beispielcode:

// Klasse für Fantasy-Figuren
class FantasyFigur {
  constructor(name, groesse, alter, laseraugen) {
    this.name = name;
    this.groesse = groesse;
    this.alter = alter;
    this.laseraugen = laseraugen;
  }

  doppelsprung() {
    return this.name + " springt doppelt so hoch!";
  }

  fliegen() {
    return this.name + " fliegt durch die Lüfte!";
  }

  laserSchneiden() {
    if (this.laseraugen) {
      return this.name + " schneidet mit Laseraugen!";
    } else {
      return this.name + " hat keine Laseraugen.";
    }
  }
}

// Objekte erzeugen
const kevin = new FantasyFigur("Kevin", 180, 25, true);
const gala = new FantasyFigur("Gala", 170, 22, false);

console.log(kevin.laserSchneiden());
console.log(gala.fliegen());
